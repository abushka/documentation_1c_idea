# documentation_1c_idea

Первоначально нужно поправить грамматические ошибки

Ошибки делятся на 6 типов

1 - Ошибки конфигурации (~172)

2 - Блокирующие (~101)

3 - Критические (~219)

4 - Значительные (~963)

5 - Незначительные (~3361)

6 - Тривиальные (~404)
